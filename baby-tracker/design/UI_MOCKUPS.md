## Design System

### Color Palette

**Primary Colors (Soft, Baby-Friendly)**
- **Primary Blue:** `#4A90E2` - Main action buttons, accents
- **Soft Teal:** `#50E3C2` - Success states, positive indicators
- **Warning Orange:** `#F5A623` - Medium severity (3/5) reactions
- **Danger Red:** `#E74C3C` - High severity (4-5) reactions
- **Info Purple:** `#9B59B6` - Informational items

**Neutral Colors (Clean, Minimal)**
- **Background:** `#FFFFFF` - Main background
- **Surface:** `#F8F9FA` - Cards, sections
- **Text Primary:** `#2C3E50` - Headings, important text
- **Text Secondary:** `#7F8C8D` - Body text, labels
- **Border:** `#ECF0F1` - Separators, outlines
- **Divider:** `#E8E8E8` - Section dividers

**Dark Mode**
- **Background:** `#1A1A1A`
- **Surface:** `#2D2D2D`
- **Text Primary:** `#FFFFFF`
- **Text Secondary:** `#B0B0B0`
- **Border:** `#3D3D3D`

### Typography

**Font:** Poppins (clean, modern, baby-friendly)

**Text Styles:**
- **H1 (Headings):** 32px, Bold, `#2C3E50`
- **H2 (Section headers):** 24px, SemiBold, `#2C3E50`
- **H3 (Card titles):** 20px, SemiBold, `#2C3E50`
- **Body (Content):** 16px, Regular, `#2C3E50`
- **Caption (Labels):** 14px, Regular, `#7F8C8D`
- **Small (Notes):** 12px, Regular, `#7F8C8D`

## Key UX Principles

### 1. Speed First
- Log a meal in <10 seconds
- One-tap actions for common tasks
- Remember last choices

### 2. Safety First
- Auto-alert family on severe reactions
- Medical disclaimer on all reaction screens
- Clear severity indicators

### 3. Family First
- Real-time sync across devices
- Invite multiple caregivers
- Shared child profiles

### 4. Clarity Over Clutter
- Clean, minimal UI
- Color-coded severity
- Progressive disclosure (show details on tap)

### 5. Tired Parent Friendly
- Dark mode for late nights
- Large touch targets
- Optional fields are truly optional
- Quick actions (FAB for meal logging)
